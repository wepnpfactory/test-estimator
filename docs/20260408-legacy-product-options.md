## 004 — 내 마음대로 제작하는 학원교재 (양장 고급지)

| --------------- | ---------------------------------------- |

### 032 — 일반지 포스터 (토너)

031과 동일, 단 `paper_size_type = A3 / B4 / 직접입력`, `print_type = 인디고·토너`.

### 033 — 일반지 포스터 (옵셋)

031 구조 + `print_type = 옵셋인쇄`, 사이즈 `A2 / A3 / B2 / B3 / B4 / 국반절 / 직접입력`, `rounded_conner = 4mm` 추가.
